Bienvenue sur ton programme RAID

Cette application t’accompagne dans ton évolution physique et mentale vers le niveau d’élite requis pour intégrer le RAID.

Programme d'entraînement

Semaine type

  • Lundi : Musculation haut du corps + course fractionnée
  • Mardi : Natation + gainage statique et dynamique
  • Mercredi : Musculation bas du corps + mobilité
  • Jeudi : Cardio (Luc Léger ou HIIT) + abdominaux
  • Vendredi : Travail technique (tractions strictes, pompes explosives)
  • Samedi : Sortie longue (5 à 10km) + étirements
  • Dimanche : Repos actif (marche, vélo doux)

Nutrition

Objectif : prise de masse légère (65 → 70kg)

  • Protéines : 2g/kg de poids de corps (≈ 130g)
  • Glucides : féculents complets (riz, avoine, patate douce)
  • Lipides : huile d’olive, oléagineux
  • Hydratation : minimum 2L/jour
